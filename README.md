## 조원 구성

- 김연준 : 팀장, 워크스페이스 생성, 조회 삭제
- 박재현 : DevOps 및 개발환경 세팅, 비동기 메신저, 칸반, 팀 공지사항 기능
- 송효원 : 게시판 생성, 조회 삭제. 일정 등록,수정,삭제
